# Style Presets — Reference Guide

A set of 13 distinct visual styles for AI website generation.

Pick one preset per generation for consistent, distinct results.

**A. Minimal / Editorial**
- Background: #FAFAF8, Text: #1A1A1A, Accent: #C9603F
- Fonts: Heading — "Playfair Display" (serif), Body — "Inter"
- Spacing: generous (100-140px section padding), thin 1px dividers
- Images: full-bleed or large single images, no shadows

**B. Bold / Modern**
- Background: #FFFFFF, Text: #0D0D0D, Accent: #5B5BFF
- Fonts: Heading — "Space Grotesk", Body — "Inter"
- Spacing: 80-100px, large type scale (h1: 4rem+)
- Shapes: geometric blocks, sharp corners, high contrast color blocks

**C. Soft / Friendly**
- Background: #FFF8F3, Text: #3A3A3A, Accent: #FF8C69, Secondary: #A3D9B1
- Fonts: Heading — "Quicksand", Body — "Nunito Sans"
- Spacing: 60-80px, border-radius 16-24px everywhere
- Imagery: illustrations or rounded photos, soft drop shadows

**D. Technical / Dark**
- Background: #0B0E14, Text: #E6E6E6, Accent: #00D9A3
- Fonts: Heading — "JetBrains Mono" or "Space Grotesk", Body — "Inter"
- Spacing: 70-90px, subtle grid-line background patterns
- Effects: glow on hover, monospace labels/tags, dark cards with 1px borders

**E. Luxury / Premium**
- Background: #0F0F0F, Text: #F5F5F0, Accent: #D4AF37 (gold)
- Fonts: Heading — "Cormorant Garamond", Body — "Lato"
- Spacing: very generous (120-160px), thin gold dividers
- Imagery: high-contrast full-width photography, minimal text overlays

**F. Retro / Vintage**
- Background: #F4ECD8, Text: #2B2118, Accent: #C1432F, Secondary: #6B7F59
- Fonts: Heading — "Abril Fatface" or "Bitter", Body — "Courier Prime"
- Spacing: 70-90px, slightly textured/paper background
- Details: muted earthy tones, thick borders, vintage badges/stamps as decoration

**G. Brutalist / Raw**
- Background: #FFFFFF, Text: #000000, Accent: #FF3300
- Fonts: Heading — "Helvetica Neue" / system sans, oversized, Body — same
- Spacing: tight, no border-radius, visible borders everywhere
- Details: harsh black borders, no shadows, unconventional grid overlaps, intentionally "unpolished"

**H. Pastel / Dreamy**
- Background: #F5F0FF, Text: #4A4458, Accent: #B8A4FF, Secondary: #FFD6E8
- Fonts: Heading — "Fraunces", Body — "Poppins"
- Spacing: 80-100px, large rounded blobs/shapes as background decoration
- Details: soft gradients, low-contrast palettes, gentle floating animations

**I. Corporate / Professional**
- Background: #FFFFFF, Text: #1F2937, Accent: #2563EB, Secondary: #F3F4F6
- Fonts: Heading — "Source Sans Pro" / "IBM Plex Sans", Body — "Inter"
- Spacing: 60-80px, structured grid layouts, clear card boundaries
- Details: subtle shadows, icon-led feature blocks, trust badges/logos row

**J. Nature / Organic**
- Background: #F7F5F0, Text: #2F3E2E, Accent: #7A9D54, Secondary: #C9A66B
- Fonts: Heading — "Fraunces" or "Lora" (serif), Body — "Karla"
- Spacing: 90-110px, asymmetric/organic shaped section dividers (curves, not straight lines)
- Details: earthy tones, leaf/texture motifs, rounded organic shapes

**K. Cyberpunk / Neon**
- Background: #08070D, Text: #E0E0E0, Accent: #FF2E97, Secondary: #00F0FF
- Fonts: Heading — "Orbitron" or "Rajdhani", Body — "Rubik"
- Spacing: 70-90px, glitch/scan-line effects optional
- Details: gradient glows, neon borders, dark cards with vibrant accent edges

**L. Monochrome / Sophisticated**
- Background: #F8F8F8, Text: #1A1A1A, Accent: #1A1A1A (single tone, no separate accent)
- Fonts: Heading — "Neue Haas Grotesk" / "Helvetica", Body — same family, different weight
- Spacing: 100-120px, heavy reliance on type-size contrast instead of color
- Details: black & white photography only, accent via scale/position not color

**M. Playful / Startup**
- Background: #FFFFFF, Text: #1E1E2E, Accent: #FF6B6B, Secondary: #4ECDC4, Tertiary: #FFE66D
- Fonts: Heading — "Poppins" (bold), Body — "Inter"
- Spacing: 70-90px, blob/squiggle decorative shapes
- Details: multi-color accent system (3 accents), rounded buttons, friendly illustrations/emoji-style icons

# Alignment & Design Theory:-
# important rules to follow for design theory
# all rules below must be fllowed before the final output

**Grid Systems**
Most modern sites use a 12-column grid. It keeps elements aligned, creates consistent spacing, and makes
responsive breakpoints easier to manage. Even creative layouts usually break from a grid intentionally, not
randomly.

**Visual Hierarchy**
Size, weight, color, and spacing tell users what to look at first. Headlines should be dramatically larger than body
text, not just slightly bigger. Contrast in size creates clear reading order: hero headline, then subheading, then
body, then CTA.

**White Space (Negative Space)**
More space around elements creates a more premium feel. Cramped layouts feel cheap; generous padding and
margins feel high-end. This is why minimalist and luxury brand sites use so much empty space.

**Alignment Principles**
Left-aligned text is most readable for body content in Western languages. Center alignment works for short
headlines, hero sections, and CTAs, but avoid centering long paragraphs. Consistent edge alignment, where text,
images, and buttons all start from the same vertical line, creates a clean, organized feel. Avoid mixing too many
alignments in one section.

**Color Theory**
The 60-30-10 rule suggests 60% dominant/neutral color for the background, 30% secondary color, and 10%
accent color for CTAs and highlights. Maintain high contrast between text and background for readability. One
accent color stands out more than three, so use it sparingly for buttons and links to draw the eye.

**Typography**
Pair a maximum of two fonts: one for headings, which can be bold or distinctive, and one for body text, which
should be clean and readable. Body text should be at least 16-18px with a line-height around 1.5-1.6 for
readability. Limit line length to roughly 60-75 characters per line for comfortable reading.

**Visual Rhythm and Repetition**
Repeating patterns, such as card layouts and consistent spacing units like 8px, 16px, 24px, and 32px, create a
sense of order. This is why spacing scales built on multiples of 8 are common in design systems.

**F-Pattern and Z-Pattern**
In the F-pattern, users scan left-to-right across the top, then down the left side; this is common for text-heavy
pages. In the Z-pattern, eyes move from top-left to top-right to bottom-left to bottom-right; this is common for
landing pages with one clear CTA.

**Mobile-First / Responsive Thinking**
Design for the smallest screen first, then scale up. Stack elements vertically, increase touch target sizes to at
least roughly 44px, and reduce unnecessary elements on mobile.

**Motion and Animation Theory**
Animations should support understanding, not just decorate; for example, a fade-in shows that new content has
appeared. Keep durations short, around 200-500ms, for UI feedback, and longer, around 800ms-1.5s, for
dramatic scroll reveals. Ease-out feels natural for elements entering, and ease-in feels natural for elements
leaving.

**Consistency**
Using the same button styles, spacing, and corner radius throughout the site is important; inconsistency is one of
the biggest signs of an unpolished site
